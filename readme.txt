Photoshop Export Workflow 0.1 Alpha
Набор скриптов для Фотошопа, позволяющий создавать сценарии экспорта файлов.

Photoshop Export Workflow (PEW) — это набор скриптов для Фотошопа, позволяющий создавать
сценарии экспорта файлов. Удобство таких сценариев в том, что можно создать сценарии
для всех вариантов экспорта (веб, экран, печать, фотобанки и т. п.) и делать экспорт
нажатием одной клавиши.

Основные возможности:
* Уменьшение файлов до заданных размеров;
* Конвертирование в профиль sRGB;
* Восстановление резкости с помощью фильтра Smart Sharpen или Unsharp Mask в канале яркости;
* Сохранение результата в JPEG заданного качества.
* Добавление рамки и подписи.
* Автоматический выбор подписи на основе поля Authour в EXIF фотографии.

Сценарии (как и сам PEW) пишутся на языке JavaScript. Вам не нужно быть программистом,
чтобы создавать простые сценарии, к тому же в дистрибутив входит много примеров готовых
сценариев, которыми можно пользоваться вообще без изменений или с минимальными изменениями.
Однако навыки программирования дадут вам огромные возможности по управлению экспортом изображений.
